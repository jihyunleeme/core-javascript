# 데이터 타입

## 01 데이터 타입의 종류

`기본형`:  `할당`, 연산시 `복제`. 값이 담긴 주솟값 바로 복제, 불변성(immutability)  
참조형 : `참조`, 값이 담긴 주솟값들로 이루어진 묶음을 가리키는 주솟값을 복제

- 기본형(원시형, primitive type)
  - 숫자 number
  - 문자열 string
  - 불리언 boolean
  - null
  - undefined
  - Symbol
- 참조형(reference type)
  - 배열 Array
  - 함수 Function
  - 날짜 Date
  - 정규표현식 RegExp
  - Map, WeakMap
  - Set, WeakSet

## 02 데이터 타입에 관한 배경지식

### 1-2-1 메모리와 데이터

- 0,1로 이루어진 하나의 메모리 조각, 비트(bit) -> 비트의 묶음 byte (1byte = 1bit)
- 메모리 주솟값을 통해 모든 데이터(바이트 단위의 식별자)로 서로 구분, 연결 짓는다.

### 1-2-2 식별자와 변수

- 변수(variable) : 변할 수 있는 무언가, 데이터
- 식별자(identifier) : 식별자
